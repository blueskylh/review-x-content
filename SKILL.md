# 审核 X 待发布内容

## 核心原则

- 把“能否发布”和“能否变现”作为两个独立结论。允许留在平台上的内容仍可能不适合变现。
- 只做发布前风险评估，不保证 X 一定批准、推荐或付费；最终执行取决于上下文、账号历史、地区法律和 X 的裁量。
- 保留作者的事实主张、立场和语气，只修改造成风险的最小范围。不要以合规为由无谓弱化合法批评、新闻报道或公共利益讨论。
- 不把账号级资格与单篇内容合规混为一谈。缺少账号数据时列为“待确认”，不得猜测。

## 先收集输入

取得或从上下文识别：

1. 完整文案、帖子串顺序、链接和话题标签。
2. 随附图片、视频、音频或截图；无法查看时要求用户描述关键画面。
3. 是否为广告、赞助、新闻报道、讽刺/模仿、成人内容或 AI/合成媒体。
4. 目标受众/地区，以及是否参与 Creator Revenue Sharing、Subscriptions 或其他变现产品。
5. 账号资格信息仅在用户要求审核收益资格时收集。

若信息不全，继续审核可见内容，并明确列出哪些结论是条件性的。

## 核验最新规则

每次进行实质审核时，优先打开 `references/policy-sources.md` 中的 X 官方页面，核对页面当前内容与生效日期。政策可能变化，不要仅凭本 Skill 中的摘要下结论。

- 只用 X 官方 Help Center、legal.x.com 或 business.x.com 作为政策结论依据。
- 如果无法联网，明确写出“基于离线清单的初步审核，发布前需复核最新官方政策”，并给出官方链接。
- 若官方页面之间存在数值或措辞差异，分别引用，指出冲突，不自行合并成确定规则。

## 审核流程

### 1. 识别内容与上下文

逐段标记文本、媒体、链接、标签和行为层面的信号。区分作者是在宣传、威胁、赞扬、报道、谴责、引用还是虚构描写；上下文会改变判断。

### 2. 审核 X Rules

至少覆盖以下类别，仅报告相关项：

- 暴力威胁、煽动、赞扬暴力及暴力/仇恨实体
- 儿童安全、自杀或自残鼓励
- 定向辱骂、骚扰、煽动围攻、基于受保护属性的仇恨行为
- 成人内容、非自愿私密影像及敏感媒体标注
- 非法或受监管商品与服务
- 个人隐私、暴露或威胁暴露私人信息
- 冒充、欺骗性身份、平台操纵、垃圾信息及虚假互动
- 选举/公民程序误导
- 可能造成伤害的合成或篡改媒体及所需披露
- 著作权、商标和未经 X 许可的视频内第三方广告

给出平台结论：

- `高风险/不建议发布`：明显触及禁止项，或仅靠标签无法修复。
- `需修改或标注`：经删除、改写、补充上下文/来源/敏感媒体或 AI 标签后可能发布。
- `低风险`：未发现明显违规，但不是平台保证。

### 3. 审核创作者变现

先应用 X Rules，再独立检查：

- 禁止变现内容：违法内容、相关产品/服务、娱乐性毒品、处方药、烟草、售酒、赌博/彩票、武器弹药爆炸物、网络盗版、垃圾或误导内容、濒危物种制品等。
- 受限变现内容：成人或性暗示、血腥/令人不适/暴力、仇恨或极端主义、灾难/冲突/大规模暴力/争议社会政治议题、粗俗或侮辱性语言。
- 行为要求：不得以违法或有害行为、实物/服务、性服务换取货币或互动；不得欺诈、虚假诱导互动或操纵平台。
- AI 生成的武装冲突视频必须清晰披露；逼真的武装冲突游戏视频也应标注为合成视频，不得伪造来源、冲突或时间。
- 账号资格：年龄、地区、账号存续、完整资料、邮箱、双重验证、良好状态、Premium、Stripe、身份验证、粉丝/展示量等。以当前官方产品页面为准；无法从草稿判断的项目写入“待确认”。

给出变现结论：

- `不符合/高风险`：属于明确禁止变现类别或明显违反行为标准。
- `受限/可能降收益`：属于受限类别，即使平台允许发布也可能限制变现或推荐。
- `内容层面低风险`：未发现明显变现限制；账号资格仍需另行确认。

### 4. 给出可执行修改

- 按风险从高到低引用具体片段，并说明触发的政策类别、原因、置信度和修复方式。
- 对事实性指控建议补充可靠来源、限定语和证据，不替用户捏造来源。
- 对隐私信息做脱敏；对威胁、仇恨、骚扰或煽动内容移除针对性和行动号召。
- 对敏感/成人/暴力或合成媒体说明应使用的 X 标签与文字披露；标签不能修复本身被禁止的内容。
- 同时提供“最小改动版”；用户要求时再给“更适合收益/推荐版”。
- 不复述不必要的个人信息、极端仇恨词或儿童性剥削材料。用占位符指出删除位置。

## 输出格式

使用用户的语言，默认采用以下结构：

```markdown
## 审核结论
- X 平台规则：低风险 / 需修改或标注 / 高风险不建议发布
- 创作者变现：内容层面低风险 / 受限或可能降收益 / 不符合或高风险
- 置信度：高 / 中 / 低

## 发现的问题
| 优先级 | 原文/位置 | 政策类别 | 判断与原因 | 修改建议 |
|---|---|---|---|---|

## 修改稿
[尽量保留原意的完整可发布版本]

## 发布前操作
- [媒体标签、AI 披露、来源、链接、账号资格等待确认项]

## 官方依据
- [当前核验过的官方政策名称、链接与核验日期]

> 这是发布前风险评估，不是 X 的正式裁定，也不是法律意见。
```

没有发现明显问题时也要简短说明已检查的高风险类别，不要虚构问题。若内容涉及当地违法性、诽谤、金融/医疗承诺或知识产权权属，只标出需要专业核验的风险，不提供确定法律结论。
